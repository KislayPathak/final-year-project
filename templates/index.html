PREDICTION OF DISEASES USING ML

The project that deals with prediction of diseases using different algorithms of Machine Learning and provides one step solution to major problems faced.

The project deals with the identification of diseases using the medical history of a person.

The model is trained with the symptoms of diseases in different persons who are diagnosed to be infected with those diseases.

A person medical history or the symptoms will be run on the trained model to depict if the person is infected with the disease or not.

Different algorithms of Machine Learning will be used to train the model for different diseases.
